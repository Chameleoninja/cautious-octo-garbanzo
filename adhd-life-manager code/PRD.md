# Product Requirements Document (PRD) for ADHD Life Manager App

## Overview
The ADHD Life Manager app is designed to help individuals with ADHD manage their tasks, time, and overall well-being. The app will provide a user-friendly interface with various features aimed at improving productivity and organization.

## Features

### 1. User Interface
- **Design**: The app will have a purple gradient design, utilizing specific shades of purple as defined in the design specifications.
- **Mobile Responsiveness**: The layout will adapt to different screen sizes, ensuring usability on mobile devices.
- **Dark Mode**: Users can toggle between light and dark themes for better visibility and comfort.

### 2. Task Management
- **Task Lists**: The app will feature three main task lists:
  - **Today**: Tasks that need to be completed today.
  - **Week**: Tasks scheduled for the week.
  - **Someday**: Tasks that are not time-sensitive but need to be tracked.
- **Subtasks**: Users can create subtasks under main tasks for better organization.

### 3. Drag and Drop Functionality
- Users can rearrange tasks within and between lists using drag-and-drop.

### 4. Modals
- **Brain Dump**: A modal for users to quickly jot down thoughts and tasks.
- **Focus Timer**: A timer to help users focus on tasks for set periods.
- **Energy Check**: A modal for users to assess their energy levels and adjust tasks accordingly.
- **Medication Log**: A log for tracking medication intake and schedules.

### 5. Time Blocking
- Users can allocate specific time blocks for tasks, with drag-and-drop functionality to adjust schedules.

### 6. Search and Filtering
- Users can search for tasks and filter them based on criteria such as due date, priority, and status.

### 7. Stats Dashboard
- A dashboard displaying live counters for completed tasks, upcoming deadlines, and other relevant statistics.

### 8. Local Storage Persistence
- The app will save user data in local storage to ensure persistence across sessions.

## Technical Specifications
- **HTML/CSS/JavaScript**: The app will be built using standard web technologies.
- **Inline CSS**: All styles will be included within the HTML file for simplicity.
- **JavaScript Libraries**: Consider using libraries for drag-and-drop functionality and modals if necessary.

## Conclusion
The ADHD Life Manager app aims to provide a comprehensive tool for individuals with ADHD to manage their daily tasks and improve their productivity. By incorporating the features outlined in this document, the app will serve as a valuable resource for users seeking to enhance their organizational skills.
