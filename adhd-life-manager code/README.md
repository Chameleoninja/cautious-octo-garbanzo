# ADHD Life Manager

## Overview
The ADHD Life Manager is a web application designed to help individuals manage their tasks and daily activities effectively. It incorporates various features tailored to support users with ADHD, including task lists, time management tools, and a user-friendly interface.

## Project Structure
```
adhd-life-manager

└── README.md
```

## Files Description

### `adhd-life-manager-pro.html`
This file contains the complete implementation of the ADHD Life Manager app. It includes:
- Inline CSS for a purple gradient design.
- JavaScript for drag-and-drop functionality.
- Modals for:
  - Brain Dump
  - Focus Timer
  - Energy Check
  - Medication Log
- Features for:
  - Time blocking
  - Subtasks
  - Search and filtering
  - Stats dashboard with live counters
  - LocalStorage persistence
  - Mobile responsiveness
  - Dark mode

This file contains the Product Requirements Document that outlines the features and specifications for the ADHD Life Manager app.

   ```
   cd adhd-life-manager
   ```
3. Open `adhd-life-manager-pro.html` in your web browser to view and interact with the application.

## Features
- **Task Management**: Organize tasks into three lists: Today, Week, and Someday.
- **Drag and Drop**: Easily rearrange tasks and time blocks.
- **Modals**: Access various tools through modals for better task management.
- **Subtasks**: Break down tasks into smaller, manageable subtasks.
- **Search and Filtering**: Quickly find tasks using search functionality.
- **Stats Dashboard**: View live counters for tasks and subtasks.
- **LocalStorage**: Persist data across sessions.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Dark Mode**: Toggle between light and dark themes for user comfort.

## Usage Guidelines
- To add a task, click on the appropriate task list and enter the task details.
- Use the Brain Dump modal to quickly jot down thoughts and tasks.
- Set timers using the Focus Timer modal to enhance productivity.
- Log medication and energy levels through their respective modals.
- Utilize the stats dashboard to monitor your progress and task completion rates.
